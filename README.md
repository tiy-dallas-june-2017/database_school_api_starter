## Postgres Database API Practice
### Getting Started
Run the following.
```
$ npm install
```

Create a file in the root of your project called .env.

Put the following in your .env file.
```
PGHOST='localhost'
PGDATABASE='database_school'
PGPASSWORD=null
PGPORT=5432
PORT=6001
```

Now run the following.

```
$ npm run initdb
```
